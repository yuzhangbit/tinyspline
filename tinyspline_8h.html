enum tsError

Contains all error codes used by TinySpline. The following code snippet shows how to handle errors: 

 tsError err = ...       // any function call here
 if (err < 0) {          // or use err != TS_SUCCESS
     printf("we got an error!");

     // you may want to reuse error codes
     return err;
 }

tsError ts_bspline_buckle ( const tsBSpline *  original,
tsReal  b,
tsBSpline *  _buckled_ 
)

Buckles spline by b and stores the result in _buckled_. Creates a deep copy of spline, if spline != _buckled_.

This function is based on: 

 Holten, Danny. "Hierarchical edge bundles: Visualization of adjacency
 relations in hierarchical data." Visualization and Computer Graphics,
 IEEE Transactions on 12.5 (2006): 741-748.

Holten calls it "straightening" (page 744, equation 1).

Usually, the range of b is: 0.0 <= b <= 1.0 with 0 yielding to a line connecting the first and the last control point (no buckle) and 1 keeping the original shape (maximum buckle). If < 0 or > 1 the behaviour is undefined, though, it will not result in an error.

On error, (and if spline != _buckled_) all values of _buckled_ are set to 0/NULL.

Parameters
splineThe spline to buckle by b.
bThe buckle factor (usually 0.0 <= b <= 1.0).
_buckled_The output parameter storing the buckled spline.
Returns
TS_SUCCESS On success.
TS_MALLOC If spline != _buckled_ and allocating memory failed.

Fills the knot vector of spline according to type with minimum knot value min to maximum knot value max and stores the result in _result_. Creates a deep copy of spline, if spline != _result_.

On error, (and if spline != _result_) all values of _result_ are set to 0/NULL.

Parameters
splineThe spline to deep copy (if spline != _result_) and whose knot vector is filled according to type with minimum knot value min and maximum knot value max.
typeHow to fill the knot vector of _result_.
minThe minimum knot value of the knot vector of _result_.
maxThe maximum knot value of the knot vector of _result_.
_result_The output parameter storing the result of this function.
Returns
TS_SUCCESS On success.
TS_DEG_GE_NCTRLP If spline->n_knots < 2*(original->deg+1). We can reuse this error code because spline->n_knots < 2*(spline->deg+1) implies spline->deg >= spline->n_ctrlp. Furthermore, using TS_DEG_GE_NCTRLP instead of TS_NUM_KNOTS ensures that TS_NUM_KNOTS is not used twice for this function. To be more fail-safe, spline->deg+1 instead of spline->order is used, to make sure that spline->deg+1 >= 1.
TS_NUM_KNOTS If type == TS_BEZIERS and spline->n_knots % spline->order != 0.
TS_KNOTS_DECR If min >= max. (ts_fequals is used to determine whether min == max).
TS_MALLOC If spline != _result_ and allocating memory failed.

tsError ts_bspline_interpolate_cubic ( const tsReal *  points,
size_t  n,
size_t  dim,
tsBSpline *  _spline_ 
)

Performs a cubic spline interpolation using the thomas algorithm, see: 

https://en.wikipedia.org/wiki/Tridiagonal_matrix_algorithm
http://www.math.ucla.edu/~baker/149.1.02w/handouts/dd_splines.pdf
http://www.bakoma-tex.com/doc/generic/pst-bspline/pst-bspline-doc.pdf

The resulting spline is a sequence of bezier curves connecting each point in points. Each bezier curve b is of degree 3 with dim being the dimension of the each control point in b. The total number of control points is (n - 1) * 4.

On error, all values of _spline_ are set to 0/NULL.

Note: n is the number of points in points and not the length of points. For instance, the follwing point vector yields to n = 4 and dim = 2: 

[x0, y0, x1, y1, x2, y2, x3, y3]
Parameters
pointsThe points to interpolate.
nThe number of points in points.
dimThe dimension of each control point in _spline_.
_spline_The output parameter storing the result of this function.
Returns
TS_SUCCESS On success.
TS_DIM_ZERO If dim == 0.
TS_DEG_GE_NCTRLP If n < 2.
TS_MALLOC If allocating memory failed.

Resizes spline by n (number of control points) and stores the result in _resized_. Creates a deep copy of spline, if spline != _result_. If back != 0 spline is resized at the end. If back == 0 spline is resized at front.

On error, (and if spline != _result_) all values of _result_ are set to 0/NULL.

Returns
TS_SUCCESS On success.
TS_DEG_GE_NCTRLP If the degree of _resized_ would be >= the number of the control points of _resized_.
TS_DIM_ZERO If spline != _result_ and spline->dim == 0.
TS_DEG_GE_NCTRLP If spline != _result_ and spline->deg >= spline->n_ctrlp.
TS_MALLOC If spline != _result_ and allocating memory failed.
